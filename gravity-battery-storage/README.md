# Gravity Battery Storage

## Problem

Lithium batteries use scarce materials (Li, Co, Ni), degrade over time, and have fire risk. Pumped hydro is efficient but limited to specific geography (you need two reservoirs at different elevations). We need cheap, long-duration energy storage that works anywhere.

## Solution

A modular gravity battery: lift heavy composite blocks (iron mine tailings + recycled concrete) up a structure with surplus electricity, lower them through generators when power is needed. Simple, scalable, lasts 50+ years, zero degradation, uses waste materials.

## Key Innovation

- **Mine tailings composite blocks** — iron ore waste compressed into 35-ton blocks, zero-cost raw material, solves mine waste problem
- **Multi-rotor kinetic recovery** — 6-arm helical drive generates on both up and down stroke
- **Self-erecting structure** — the battery builds its own tower as it operates, using the first blocks lifted as the base
- **90% round-trip efficiency** — regenerative braking + low-friction maglev guide rails
- **Infinite cycle life** — no chemical degradation, mechanical system lasts 50+ years with basic maintenance

## Target Cost

$50/kWh (vs $150/kWh lithium), $0.001/kWh marginal cost per cycle

## Impact
- Cheapest grid-scale storage at any duration (1 hour to 100+ hours)
- Zero scarce materials — iron, concrete, gravity
- No fire risk, no toxic chemicals, no thermal runaway
- Works in any geography — flat desert, urban basement, abandoned mine shaft
- Can be built incrementally — add blocks and height as needed
